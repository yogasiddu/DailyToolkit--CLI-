# 🧰 DailyToolkit

一个轻量、实用的日常工具合集命令行程序（CLI）。  
无论是快速记录、查天气、看 IP、管理待办，这个工具都能帮到你。

## ✨ Features

- 🌤 天气查询（支持城市名称）
- 📝 轻量记事本（本地保存）
- ✅ 待办事项管理
- 🌐 IP 信息查询（公共 IP / 地区）
- 🕒 日期与时间小工具
- 🎛 可扩展模块化设计

## 📦 安装

```bash
git clone https://github.com/你的用户名/DailyToolkit
cd DailyToolkit
pip install -r requirements.txt
